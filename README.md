# auto-mowers-nodejs

# Install
```bash
  git clone https://github.com/fablanglet/auto-mowers-nodejs
  npm install
```

# Start Unit Test
```bash
  npm test
```
or launch test to read file (filename: testfile)
```bash
  mocha tests/start_mowers.js
```

# Start NodeJS Express
```bash
  npm start
```

## End point
```bash
  GET http://localhost:3000/launch
 ```
